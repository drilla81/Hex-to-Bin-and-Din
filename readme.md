## What is a Power?

A **power/exponent** is just repeated multiplication.

Example:

2^3=2\times2\times2=8

Read it like:

> “2 to the power of 3”

Meaning:

```text
Multiply 2 by itself 3 times
```

---

# The Two Important Parts

| Part             | Name           | Example |
| ---------------- | -------------- | ------- |
| Big number       | Base           | 2       |
| Small top number | Exponent/Power | 3       |

So in:

```text
2³
```

* 2 = base
* 3 = exponent

---

# Funny Real-Life Meaning

## Exponents are like cloning machines.

```text
2² = 4
2³ = 8
2⁴ = 16
```

The number keeps multiplying itself like:

> “Me and my copies are starting a business.”

# SUPER IMPORTANT SHORTCUTS

## Anything power 1

a^1=a

Example:

```text
5¹ = 5
```

Because:

> “One copy only. No cloning today.”

---

## Anything power 0

a^0=1

Example:

```text
999⁰ = 1
```

Math basically says:

> “Everybody calm down. The answer is 1.”

---

## Power 2 = Square

```text
5² = 25
```

Called a **square** because math people love geometry too much.

---

## Power 3 = Cube

```text
2³ = 8
```

Called a cube because apparently squares weren’t enough.
